# 카드 아카이브 — 모바일 카드 수집 게임

Pokemon TCG 스타일의 3D 카드 팩 개봉 + 컬렉션 게임. 모바일/데스크탑 양쪽 모두 작동합니다.

## 기능

- 🎴 3D 카드 팩 드래그 뜯기
- ✨ 레어도별 홀로그래픽 효과 (커먼/언커먼/레어/울트라 레어)
- 📸 직접 사진을 올려 나만의 카드 만들기
- 🎲 카드별 등장 확률(가중치) 조절
- 📊 컬렉션 진행도 표시
- 💎 울트라 레어 등장 시 화려한 이펙트

## 데이터 저장

모든 데이터는 사용자의 브라우저 `localStorage`에 저장됩니다. 다른 사용자와 컬렉션이 공유되지 않습니다.

## 사용 방법

화면 위쪽에서 팩을 가로로 드래그해서 절반 이상 뜯으면 자동으로 열립니다. 카드가 펼쳐지면 탭으로 다음 카드, 드래그로 각도 조절이 가능합니다.

## 호환성

- Android Chrome ✅
- iOS Safari ✅
- 데스크탑 Chrome/Firefox/Edge ✅
- WebGL 2.0 지원 필수

- Three.js 0.160 (WebGL 3D 렌더링)
